    Pairs()

    Call this function instead of using pairs.

    Declaration
    [LuaMethod("__pairs")]
[LuaMethod("Pairs")]
public object[] Pairs()
    Returns
    Type Description
    Object[]

    Multiple values: the next function that iterates the object, the object/table itself and nil
    Remarks

    The __pairs metamethod is implemented, but won't work in Garry's Mod. If Garry's Mod updates to a version >= Lua 5.2 it will. Read more about pairs in the Lua manual

    Examples

    You can loop over the values in a BSON Document like so:

    for key, value in bsonDocument:Pairs() do
    print(key, value)
end
